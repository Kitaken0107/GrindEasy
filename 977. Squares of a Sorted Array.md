# 2nd~3rd
2乗の部分の表現を簡素化した

```python
class Solution:
    def sortedSquares(self, nums: List[int]) -> List[int]:
        for i in range(len(nums)):
            nums[i] *= nums[i]
        
        return sorted(nums) 
```
