leetcode two sum   
https://leetcode.com/problems/two-sum/   

1st   
i,jの2変数でブルートフォース   


```python
class Solution:
    def twoSum(self, nums: List[int], target: int) -> List[int]:
    
        for i in range(len(nums)):
            for j in range(i+1,len(nums),1):
                if target == nums[i] + nums[j]:
                    return [i,j]
                    break
```
