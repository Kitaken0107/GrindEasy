# 13. Roman to Integer
https://leetcode.com/problems/roman-to-integer/

# 1st
基礎になるRomanとintのテーブルを作成   
IXCが来たときはその後ろの文字を確認して、   
後ろの方が大きければ、順序を逆さにして引き算する

```python
class Solution(object):
    def romanToInt(self, s):
        """
        :type s: str
        :rtype: int
        """
        base_roman_int = {'I':1,
                       'V':5,
                       'X':10,
                       'L':50,
                       'C':100,
                       'D':500,
                       'M':1000}
        roman_int = 0
        i = 0
        
        while i <= len(s) - 1:
            if i != len(s) - 1:
                if s[i] == 'I' and s[i+1] in ['V','X']:
                    roman_int += base_roman_int[s[i+1]] - base_roman_int[s[i]]
                    i += 2
                    continue
                elif s[i] == 'X' and s[i+1] in ['L','C']:
                    roman_int += base_roman_int[s[i+1]] - base_roman_int[s[i]]
                    i += 2
                    continue
                elif s[i] == 'C' and s[i+1] in ['D','M']:
                    roman_int += base_roman_int[s[i+1]] - base_roman_int[s[i]]
                    i += 2
                    continue
            roman_int += base_roman_int[s[i]]
            i += 1
            
        return roman_int
```

# 2nd・3rd
### 主な変更点
テーブル名が長いので、baseに変更   
ロジックとして、特定の文字の際に右側の値より小さければ、その文字は引けば良い   

```python
class Solution(object):
    def romanToInt(self, s):
        base = {'I':1,
                 'V':5,
                 'X':10,
                 'L':50,
                 'C':100,
                 'D':500,
                 'M':1000}
            
        roman_int = 0
        for i in range(len(s)):
            if i < len(s) - 1 and base[s[i]] < base[s[i+1]]:
                roman_int -= base[s[i]]
            else:
                roman_int += base[s[i]]
        return roman_int
```
